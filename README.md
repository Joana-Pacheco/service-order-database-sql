# 💻 Service Order Database (SQL)

Projeto de banco de dados relacional desenvolvido utilizando MySQL.

## 📌 Objetivo
Modelar e implementar um sistema de ordem de serviço, incluindo:
- Clientes
- Técnicos
- Serviços
- Ordens de serviço
- Finalização

## 🧱 Modelagem
O projeto foi modelado utilizando MER (Modelo Entidade-Relacionamento).

📁 Arquivo disponível: MER.png

## ⚙️ Tecnologias
- Draw.io
- MySQL
- MySQL Workbench

## 🗄️ Estrutura
O banco de dados contém:
- Tecnico
- Cliente
- Servico
- OrdemServico
- Finalizacao
- OrdemServico_Servico

## 🔍 Consultas
O projeto inclui:
- Contagem de serviços
- Ordens abertas
- Serviços por ordem
- Total por ordem (GROUP BY)

## 🚀 Como executar
1. Execute `create_table.sql`
2. Execute `inserts.sql`
3. Execute `queries.sql`
